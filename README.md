# Лабораторная работа №24-26 - Современный JavaScript
**Выполнили:** Чешуина А.Д., Антонова П.А.  
**Группа:** ИСП-231 
**Дата:** 15.04.2026
## Цель работы
Изучить и закрепить ключевые возможности современного JavaScript (ES6+):
- Деструктуризация
- Spread/Rest операторы
- Модули (Import/Export)
- Промисы и Async/Await
- Fetch API для работы с REST API
- Optional Chaining и Nullish Coalescing
## Основные выводы
В ходе выполнения лабораторных работ были изучены и применены на практике ключевые возможности современного JavaScript:
1. **Деструктуризация** значительно упрощает работу с объектами и массивами
2. **Spread/Rest операторы** позволяют легко объединять и копировать массивы и объекты
3. **Модули** помогают организовать код в больших проектах
4. **Async/Await** упрощают работу с асинхронным кодом
5. **Fetch API** - современный стандарт для HTTP-запросов
6. **Optional Chaining** предотвращает ошибки при работе с вложенными объектами
